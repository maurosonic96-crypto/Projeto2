# ☁️ cloud-so-app

Projeto acadêmico desenvolvido na disciplina de Sistemas Operacionais do curso de Análise e Desenvolvimento de Sistemas da Fatec Itapetininga.

A aplicação utiliza Node.js, Express e Render para monitoramento de informações do sistema operacional em ambiente local e cloud, permitindo analisar recursos computacionais e compreender, na prática, conceitos relacionados à infraestrutura moderna de software.
Link do projeto:https://projeto2-uz10.onrender.com

---

# 🎯 Objetivo Acadêmico

O principal objetivo do projeto foi aplicar conceitos de Sistemas Operacionais em um cenário real de desenvolvimento backend e computação em nuvem.

A proposta evoluiu de uma API simples para uma aplicação capaz de:

- monitorar recursos do sistema operacional;
- interpretar métricas computacionais;
- identificar diferenças entre ambiente local e cloud;
- simular conceitos relacionados ao gerenciamento de processos e infraestrutura.

Essa evolução elevou o projeto para um nível mais próximo de aplicações reais de monitoramento e engenharia de sistemas.

---

# 🚀 Evolução do Projeto

## 📌 Versão Inicial

A aplicação realizava:
- exibição básica de informações do sistema;
- leitura de CPU, memória e uptime;
- deploy simples em ambiente cloud.

---

## 📌 Versão Atual

A nova versão passou a incorporar:
- métricas interpretáveis;
- monitoramento mais detalhado;
- análise de ambiente cloud;
- arquitetura mais organizada;
- visão integrada entre hardware, sistema operacional e backend.

Essa mudança representou uma evolução importante do ponto de vista acadêmico, pois o projeto deixou de apenas exibir dados e passou a interpretar e estruturar informações do ambiente computacional.
